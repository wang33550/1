# v0.1.1

## 概要

`v0.1.1` 把项目从“本地概念验证”推进到了“可以开源给开发者试用”的阶段。

## 主要变化

- 新增外部宿主会话的标准化 trace 导入与回放
- 新增 OpenAI / Anthropic 的 live replay
- 新增 Codex / Claude 本地 trace 采集与脱敏
- 新增基于 harvested traces 的 corpus 级评测
- 新增第一版真实 trace 效果报告，显示 runtime 在 `next_action` 连续性上优于简单总结
- 新增 GitHub CI、Issue 模板、PR 模板与 trace 贡献说明

## 实际结果

在仓库内整理过的真实 trace 语料上：

- `runtime` 在 `6/6` 个场景里保持了 `100%` 的 `next_action` 恢复率
- `simple_summary` 降到了 `66.7%`
- 在高保真的 Codex 子集上，`runtime` 为 `100%`，`simple_summary` 降到 `33.3%`

- 包版本：`0.1.1`
