# v0.1.1

## Summary

`v0.1.1` turns the project from a local MVP into a usable open-source developer release.

## Highlights

- added normalized trace import and replay for external host sessions
- added live replay support for OpenAI and Anthropic models
- added Codex and Claude local trace harvesting with redaction
- added corpus-level replay over harvested real traces
- added the first real-trace evaluation report showing stronger next-action continuity than naive summarization
- added GitHub CI, issue templates, PR template, and trace contribution guidance

## Practical Result

On the curated harvested real corpus in this repository:

- `runtime` kept `100%` next-action recovery on `6/6` scenarios
- `simple_summary` dropped to `66.7%`
- on the higher-fidelity Codex-only slice, `runtime` kept `100%` while `simple_summary` dropped to `33.3%`

## Upgrade Notes

- package version is now `0.1.1`
- CLI version is now `0.1.1`
- use `npm run release:check` before publishing future releases
